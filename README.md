# **Fridge‑Friend 🧊**

![Vite](https://img.shields.io/badge/Vite-FF4655?style=for-the-badge&logo=vite&logoColor=white)

**Fridge‑Friend** is a smart web app that suggests recipes based on ingredients you already have in your fridge.

---

## **Demo**
[Live Version](https://fridge-friend-chi.vercel.app/)

---

## **Built With**
- Vite  
- React / Vanilla JS  
- Gemini API  
- CSS / Tailwind CSS  
- Vercel  

---

## **Getting Started**
```bash
# Install dependencies
npm install

# Create .env.local and add:
VITE_GEMINI_API_KEY=YOUR_API_KEY_HERE

# Run development server
npm run dev

# Build for production
npm run build

---

## **Usage**
Open the app (http://localhost:3000)

Enter your fridge ingredients

Click Find Recipes

Browse suggestions or save favorites
